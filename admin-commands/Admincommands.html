ADMIN COMMANDS

1. # <command> &:

To run a command in the background

2. $ sudo chgrp :

        To change group of a file

3. # chown:

        change file owner and group

r3ddv@r3ddv-Predator-G3-571:~/deviceDrivers$ sudo chown

        

4.  # apt:

apt (Advanced Package Tool) is the command-line tool for handling packages. It provides a command-line interface for the package management of the system.

        r3ddv@r3ddv-Predator-G3-571:~/deviceDrivers$ sudo apt-get install <package-name>

5. # bash:

        To get root access.

        

6. # umount :

        To unmount filesystems

7. # hwclock:

        Query and set hardware clock

8. # sudo --version:

         Print the sudo version string as well as the version string of the security policy plugin and any I/O plugin.

9. # partx  :- Tell the Linux kernel about the presence and numbering of on-disk partitions.

10. # fsck:

fsck is used to check and optionally repair one or more Linux file systems. 

        

11. # kbdrate - reset the keyboard repeat rate and delay time

12. # slabtop - display kernel slab cache information in real time

13. # runuser - run a shell with substitute user and group IDs, similar to su.

14. # visudo: For opening sudoers file to add new rules.

15. # sudo !!: Runs the previous command using sudo.

USER COMMANDS:

        

1. $ gzip [parameters] <files(s)> :

        This program compresses the contents of files.

2. $ kill <process-ID>: Kills the process of the given PID.

3. $ nmap :

        scans a server to locate open ports and the services associated with those ports.

4. $ watch :

        Run a command repeatedly , at specified time intervals.

5. $ df :

        shows filesystem usage in tabular form.

6. $         modinfo :

        output information about a loadable module

7. $ cron - daemon to execute scheduled commands (Vixie Cron)

8. $ rev - reverse lines of a file

9. $ jpegtran - lossless transformation of JPEG files

10. $ pdftohtml - program to convert pdf files into html, xml and png images

11. $ pinky - lightweight finger

12. $ pmap - report memory map of a process

13. $ ifconfig - configure a network interface

14. $ md5sum - compute and check MD5 message digest.

15. $ du :        Show disk usage , recursively
